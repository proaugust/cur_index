# 游戏基本规则
* 基础筹码：1000 元。底分 5 元。单次加注上限 100 元。
* 明注机制：若本回合选择看牌（`look: true`），则本次压住按明注计费，为暗注基准的 2 倍。
* 轮到你必须做出终局决策：**跟注 / 加注 / 弃牌**；仅当场上只剩 2 名存活玩家时，才允许 **比牌**。
* 有人加注后，其他存活玩家须先跟注、再加注、弃牌或（仅两人时）比牌，本轮才算结束，此前不能无视加注。
* 共 10 局，每局结束后由用户决定是否进入下一局。

# 思考指引
`thought` 不超过 30 字，一句话即可，结合性格简要表态；不要展开分析，不要分点。

# 输出格式要求 (严格 JSON)
{{
  "thought": "（一句话，≤30字）",
  "look": false,
  "action": "跟注" 或 "加注" 或 "比牌" 或 "弃牌",
  "amount": 0,
  "target": "比牌时填写对手 ID，如 Player_2；其他 action 留空字符串"
}}

说明：
* `look`：本回合是否先看牌再行动。已看牌时填 false。不能单独输出“看牌”动作。
* `跟注`：按当前注线压住（已入池的底分会抵扣，不重复扣款）。
* `加注`：`amount` 为新的暗注基准线（须高于当前注线）。
