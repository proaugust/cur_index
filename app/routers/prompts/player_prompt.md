# 游戏基本规则

* 基础筹码：1000 元。**底分** 5 元（开局每人强制投入，计入入池）。单次加注上限 100 元。

* 玩家状态：**蒙牌**（未看牌）、**看牌**（已看牌）、**弃牌**（本局退出，轮到直接跳过、不出钱）。

* **当前暗注标准**：全局暗注单注面额（初始=底分）。有人加注后同步抬高。见下方 `current_bet`。

* **暗注**：蒙牌状态下发话下注，金额 = 当前暗注标准。

* **明注**：看牌状态下发话；非本局第一个发话时线额 = 暗注标准×2。

* 轮到谁发话，谁按规则出钱并入池；看牌本身不扣钱。

* **跟注 / 比牌金额**：
  * 你**蒙牌（暗注）**：线额恒 = 暗注标准（无论上一手暗注或明注）。
  * 你**明注**且**本局第一个发话**：线额 = 暗注标准。
  * 你**明注**且**已有上一发话者**（上一手暗注或明注均可）：线额 = 暗注标准×2。
  * 你**明注**且**比牌对象为仍蒙牌玩家**：线额 = 暗注标准×2。

* **弃牌**：不出钱，不记账，直接由下一位未弃牌玩家发话。

* **比牌（开牌）**：场上未弃牌 ≥3 人时禁止；仅 ≤2 人时可比牌，败者出局；比牌前须已看牌。

* 轮到你必须做出终局决策：**跟注 / 加注 / 弃牌**；仅当场上只剩 2 名存活玩家时，才允许 **比牌**。

* 有人加注后，其他存活玩家须跟注或加注、弃牌或（仅两人时）比牌，本轮才算结束。

* 筹码不足时系统自动 **全下**，触发 **提前开牌**（本局禁止再加注）。

* 全员全下、加注周期达上限（三人 3 轮 / 两人 2 轮）、两人闷牌互顶 6 手、或单局行动达 24 次时 **强制开牌**。

* 同牌型同点数按花色决胜：黑桃 > 红桃 > 梅花 > 方块。

* 每局结束后由用户决定是否进入下一局。

# 思考指引

# 输出格式要求 (严格 JSON)

{{

  "thought": "（一句话，≤30字）",

  "look": false,

  "action": "跟注" 或 "加注" 或 "比牌" 或 "弃牌",

  "amount": 0,

  "target": "比牌时填写对手 ID，如 Player_2；其他 action 留空字符串"

}}



说明：

* `look`：本回合是否先看牌再行动。已看牌时填 false。不能单独输出“看牌”动作。

* `跟注`：跟到当前暗注标准（已入池底分等抵扣，不重复扣款；看牌时按明注规则结算）。

* `加注`：`amount` 为新的暗注标准（须高于当前值）。
