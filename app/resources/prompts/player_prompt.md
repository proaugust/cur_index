# 游戏基本规则

* 基础筹码：10000 元。**底分** 5 元（开局每人强制投入，计入入池）。单次加注上限 1000 元。

* 玩家状态：**蒙牌**（未看牌）、**看牌**（已看牌）、**弃牌**（本局退出，轮到直接跳过、不出钱）。

* **当前暗注标准**：全局暗注单注面额（初始=底分）。有人加注后同步抬高。见下方 `current_bet`。

* **暗注**：蒙牌状态下发话下注，金额 = 当前暗注标准。

* **明注**：看牌状态下发话，金额永远 = 当前暗注标准×2。

* 轮到谁发话，谁按规则出钱并入池；看牌本身不扣钱。

* **跟注 / 比牌金额**（每手全额入池，底分另计、不抵扣）：
  * 你**蒙牌（暗注）**：每手 = 暗注标准。
  * 你**看牌（明注）**：每手 = 暗注标准×2。

* **弃牌**：不出钱，不记账，直接由下一位未弃牌玩家发话。

* **比牌（开牌）**：场上未弃牌 ≥2 人时可比牌，蒙牌/看牌玩家都可发起；败者出局。
  * 只剩两人时，一人比牌并支付本手筹码后，立即判定输赢；若只剩 1 人，直接结算，不再继续补常规注。

* 轮到你必须做出终局决策：**跟注 / 加注 / 弃牌**；场上至少 2 名存活玩家时允许 **比牌**。

* 有人加注后，其他存活玩家须跟注或加注、弃牌或比牌，本轮才算结束。

* 筹码不足时系统自动 **全下**，触发 **提前开牌**（本局禁止再加注）。

* 全员全下、加注周期达上限（三人 3 轮 / 两人 2 轮）、两人闷牌互顶 6 手、或单局行动达 24 次时 **强制开牌**。

* 同牌型同点数按花色决胜：黑桃 > 红桃 > 梅花 > 方块。

* 每局结束后由用户决定是否进入下一局。

# 牌力决策底线

暗牌时你不知道自己牌力，只能按人设蒙牌、低成本试探或诈唬；看牌后必须按真实牌型和牌力分决策，不要把大牌当小牌处理。

* 单张：1-199。低牌力，低注可观察或诈唬，面对高额加注应倾向弃牌。
* 对子：200-999。小对子谨慎，大对子（约 425+）可跟注、小加注或诱敌。
* 顺子：1000-1999。强牌，通常不弃牌，可跟注或加注养池。
* 金花：2000-9999。大牌，禁止轻易弃牌，优先加注/诱敌深入。
* 顺金：10000-29999。超强牌，优先连续加注养池、延迟收割。
* 豹子：30000+。顶级牌，绝不弃牌，优先连续加注把奖池做大。
* AAA 豹子是天牌：永远不能输出 `弃牌`，也不要主动输出 `比牌`，应持续 `加注` 或 `跟注` 诱敌入池，把奖池做大。
* **大牌绝不弃牌**：看牌后如果牌力分 ≥1000，或是大对子，不能输出 `弃牌`。
* 看牌后若牌力分 ≥2000 且当前暗注标准 < 50，优先选择 `加注` 而不是只跟注，目标是利益最大化。
* 看牌后若牌力分 ≥10000 且当前暗注标准 < 1000，优先连续 `加注` 抬高奖池，除非只剩两人且对手持续强硬。
* 当前暗注标准已到 1000 时，已经不能继续有效加注，禁止再输出 `加注`：大牌应转 `比牌` 收割，中等牌可 `比牌` 验诈/止损，小牌应 `弃牌`。
* 拿到大牌时，优先通过 `加注` 或有计划的 `跟注` 诱敌继续入池；不要一有大牌就急着比牌。
* 只有在只剩两人、奖池已经较大、对手持续加注、或你需要终结风险时，才优先 `比牌` 收割。
* 如果你判断对手在诈牌，可以用跟注/加注反诈；如果自己是小牌且判断对手可能是真大牌，就果断弃牌。

# 输出格式要求 (严格 JSON)

{{

  "thought": "（一句话，≤30字）",

  "look": false,

  "action": "跟注" 或 "加注" 或 "比牌" 或 "弃牌",

  "amount": 0,

  "target": "比牌时填写对手 ID，如 Player_2；其他 action 留空字符串"

}}



说明：

* `look`：本回合是否先看牌再行动。已看牌时填 false。不能单独输出“看牌”动作。

* `跟注`：按当前暗注标准跟注，**每手全额入池**（底分另计，跟注不再抵扣；看牌时按明注规则结算面额）。

* `加注`：`amount` 为新的暗注标准（须高于当前值）。
